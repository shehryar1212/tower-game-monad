##  About

**Tower Game Monad** is a browser based tower stacking game built using TypeScript and Vite. The goal of the game is to stack moving blocks precisely to build the tallest tower possible.

This project showcases interactive UI design, game logic implementation in TypeScript, and modern frontend tooling.
